# *senSPure*&trade; SDK

The *senSPure*&trade; SDK (Software Development Kit) is supported multiple operating systems, which facilitates application development.  
It provides the ToF Viewer as a sample code for evaluation and development. This allows users to easily test camera performance and quickly transition into development.

# Supported Platform
* Windows: Windows 10/11
* Linux: Ubuntu 20.04/22.04
* NVIDIA Jetson: Jetpack 5.0.1/6.0

# Supported Devices
* *senSPure*&trade; C11U

# Folder description
* **windows_x64**

    The ToF viewer for the x64 (x86-64) platform on Windows 10/11.

* **ubuntu_x64**

    The ToF viewer for the x64 (x86-64) platform on Ubuntu 20.04 / 22.04.

* **jetson_arm64**

    The ToF viewer for the arm64 platform on NVIDIA Jetson AGX Orin.
